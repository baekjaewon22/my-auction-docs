## 카테고리 3. 컨설턴트 일지 시스템

### [3-1~3-2] 현장출퇴근 자동화
- 입찰/미팅/임장 09:00 시작 → 현장출근 자동 체크
- 입찰/미팅/임장 18:00 종결 → 현장퇴근 자동 체크
- 대리입찰 시 자동 비활성화

### [3-3] 일정 공백 알림
- 09:00~18:00 (휴게시간 제외) 빈 구간 감지 → 대시보드 알림

### [3-4] 전체이력 현장출퇴근 표시
- 월간 뷰에 초록점(출근)/빨간점(퇴근) 표시 + 호버 팝업 배지

### 일지 폼 개선
- 물건 → 물건번호로 명칭 변경 + 사이즈 확대
- 고객명 → 계약자명으로 일괄 변경
- 입찰: 계약자명/입찰자명 분리 (기본값=계약자명)
- 임장: 계약자명 다음줄로 레이아웃 변경

## 카테고리 5. 일지 vs 보고서 교차검증

### [5-1] 일지-보고서 교차검증

| 일지 유형 | 매칭 보고서 |
|-----------|-------------|
| 입찰/임장/미팅 (외근) | 외근 보고서 |
| 개인 (연차/월차) | 연차휴가 신청서 |
| 개인 (반차) | 반차 신청서 |
| 개인 (시간차) | 지각/조퇴/외출 사유서 |
| 개인 (병가) | 결근 사유서 |
| 미팅 (출장) | 출장 신청서 + 보고서 |

### [5-2] D-Day 대시보드 알림
- 담당자: "내 미제출 보고서" D+N 경과일 상시 표시
- 관리자: 전 직원 미제출 알림 + D-Day 표시
- 보고서 제출 시 자동 소멸

## 연차 관리 시스템

### 입사기준일
- 사용자관리 > 상세페이지에서 입사기준일 입력 (대표/회계 권한)
- 입사기준일 기준 연차 자동 계산 (1년 미만: 월차, 1년 이상: 연차 15일+)
- 2026-03-01 이전 입사자 → 2026-03-01 기준 월차 기산

### 시간차 신청 추가
- 휴가 신청 폼에 시간차 유형 (1~7시간 선택)
- 연차에서 시간 비례 차감 (예: 4시간 = 0.5일)

### 유급 계산기/환급금 오픈
- master/ceo/cc_ref 권한에 예상환급금 카드 + 환급 계산 탭 표시

### 무급휴가 → 급여정산 반영
- leave_requests 테이블 마이그레이션 적용
- 무급휴가 승인 시 급여정산에 자동 공제
- 급여 0원이어도 무급휴가 일수 표시

### 사용자관리 연차관리 탭 삭제 (기능 이전 완료)

## 카테고리 6. 매출 관련 기능

### [6-1] 계약내용 기록
- 계약 유형 선택 시 감정가%/낙찰가% 입력란 (소수점 1자리, 필수)
- 계약조건 기록 목적

### [6-2] 계약서 제출/미제출 체크

| 상태 | 표시 | 색상 | 처리 |
|------|------|------|------|
| 미확인 | 제출/미작성 버튼 | - | 담당자 클릭 |
| 제출 클릭 | 확인 대기 | 파랑 | 관리자/회계 확인 |
| 확인 완료 | 제출 | 초록 | 최종 |
| 미작성 + 사유 | 미작성 | 빨강 | 관리자/회계 승인 |
| 승인 완료 | 미제출 | 주황 | 최종 |

- 모든 단계 confirm 팝업
- 회계/마스터: 취소 기능 (작은 밑줄 텍스트)
- 대시보드에 "계약서 확인 대기" 알림 섹션

### [6-3] 활동내역
- 매출확인 > 활동내역 탭
- 담당자 Select → 계약자 리스트 (20명 페이징)
- 마지막 등록일 / 고객명 / 낙찰여부 표시
- 클릭 → 상세 타임라인 (매출+일지 통합, 날짜순)

### [6-4] 엑셀 업로드
- 양식: 담당자 / 날짜 / 고객명 / 매출항목 / 입금액
- 양식 다운로드 버튼
- 담당자명 → user_id 자동 매칭
- 확정매출로 등록 → 회계장부+급여정산 자동 반영
- 누락 건 사유 표시

### 매출 리스트 정렬
- 모든 컬럼 헤더 클릭으로 오름차순/내림차순 정렬
