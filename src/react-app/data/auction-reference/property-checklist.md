# 물건별 체크리스트

> 마이옥션 물건종류와 동일한 분류를 사용합니다. 모든 물건은 **공통 기본점검**을 먼저 적용하고, 해당 물건종류의 추가점검을 함께 적용합니다.
> 기준일 2026-07-13. 실제 입찰 전 등기부·매각물건명세서·현황조사서·감정평가서 원문을 다시 확인해야 합니다.

## 공통 기본점검

- [ ] 사건번호, 법원, 물건번호와 소재지가 입력자료와 일치하는지 확인
- [ ] 등기부, 매각물건명세서, 현황조사서, 감정평가서를 모두 확보했는지 확인
- [ ] 말소기준권리와 매각으로 소멸하지 않는 인수권리를 확인
- [ ] 임차인의 전입일·사업자등록일, 확정일자, 배당요구일을 상호 대조
- [ ] 선순위 임차인·전세권 및 보증금 인수 가능성을 확인
- [ ] 당해세, 임금채권, 최우선변제금 등 선순위 배당항목을 확인
- [ ] 감정가·최저가·예상배당·선순위채권·집행비용을 대조
- [ ] 취하·무잉여·변경 가능성과 입찰기일 직전 문건접수 내역을 확인
- [ ] 점유자, 명도 난이도, 예상 명도기간과 비용을 확인
- [ ] 관리비·공과금·체납금 및 낙찰자 부담 가능성을 확인
- [ ] 현장 사진·지도·공부상 현황과 실제 이용상태가 일치하는지 확인
- [ ] 제시외 물건, 매각 제외 물건, 공유지분 및 법정지상권 가능성을 확인

## 주거용 부동산

### 아파트
- [ ] 동·호수·전유면적·대지권 비율이 등기부와 감정평가서에서 일치하는지 확인
- [ ] 관리비 체납과 공용부분 관리비 인수 가능성을 확인
- [ ] 재건축·리모델링·조합원 지위·추가분담금 여부를 확인
- [ ] 동일 단지 최근 실거래가와 최저가를 비교

### 주택
- [ ] 토지와 건물 소유자가 동일하고 모두 매각대상인지 확인
- [ ] 제시외 건물·불법 증축·무단 용도변경 여부를 확인
- [ ] 진입로, 접도, 주차 및 실제 점유자를 확인

### 다세대(빌라)
- [ ] 호수·전유면적·대지권이 등기부와 건축물대장에서 일치하는지 확인
- [ ] 대지권 미등기, 토지 별도등기 및 위반건축물 여부를 확인
- [ ] 보증금이 시세에 근접한 깡통전세·임차권등기 위험을 확인

### 다가구주택
- [ ] 전체 임차인을 세대별로 정리하고 전입일·확정일자·보증금을 대조
- [ ] 세대별 배당순위와 인수될 보증금 합계를 계산
- [ ] 다수 점유자에 따른 명도기간과 비용을 반영

### 근린주택
- [ ] 주택과 근린생활시설의 실제 사용면적 및 공부상 용도를 구분
- [ ] 주택임대차보호법과 상가건물임대차보호법 적용 임차인을 각각 분석
- [ ] 불법 용도변경, 부가세 및 영업 인허가 문제를 확인

### 오피스텔
- [ ] 주거용·업무용 실제 사용형태와 전입·사업자등록 여부를 확인
- [ ] 적용 임대차보호법과 선순위 보증금 인수 여부를 확인
- [ ] 관리비·주차비·부가세와 대지권 누락 여부를 확인

### 도시형생활주택
- [ ] 건축물대장상 도시형생활주택 종류와 실제 사용상태를 확인
- [ ] 세대별 전입·보증금·임차권등기 및 관리비 체납을 확인
- [ ] 주차장 기준, 불법 확장과 대지권 비율을 확인

## 상업용 부동산

### 근린시설
- [ ] 건축물대장상 근린생활시설 세부 용도와 실제 영업이 일치하는지 확인
- [ ] 임차인의 사업자등록·확정일자·환산보증금과 대항력을 확인
- [ ] 영업 인허가, 위반건축물 및 원상복구 위험을 확인

### 근린상가
- [ ] 상가임대차보호법 적용과 임차인 배당·인수 여부를 확인
- [ ] 권리금·인테리어·유치권 주장 및 명도 저항 가능성을 확인
- [ ] 상권, 공실률, 관리비와 실제 영업 가능성을 확인

### 상가
- [ ] 호실 경계·전용면적·공용면적·대지권이 실제 사용과 일치하는지 확인
- [ ] 관리단·상가번영회·주차권·간판권 등 별도 부담을 확인
- [ ] 임차인의 대항력, 시설권리금과 관리비 체납을 확인

### 공장
- [ ] 공장등록 업종과 실제 운영 업종 및 입주자격을 확인
- [ ] 공장저당 기계기구 목록과 현존 설비·매각 포함 여부를 대조
- [ ] 폐수·위험물·소방·환경 인허가와 폐기물 처리비용을 확인

### 아파트형공장
- [ ] 지식산업센터 입주 가능 업종과 실입주·임대·전매 제한을 확인
- [ ] 관리기관 입주계약 승계와 사업자등록 요건을 확인
- [ ] 관리비·주차권·지원시설 사용권 및 임차인 대항력을 확인

### 숙박시설
- [ ] 숙박업 인허가 승계와 소방·피난시설 기준을 확인
- [ ] 객실 집기·비품·예약자산의 매각 포함 여부를 확인
- [ ] 위반건축물, 리모델링 유치권과 다수 점유자 명도 위험을 확인

### 주유소
- [ ] 위험물·석유판매업 인허가 승계와 시설검사 여부를 확인
- [ ] 토양오염 검사, 유류탱크 철거 및 정화비용을 확인
- [ ] 영업시설·세차장·캐노피 등 제시외 물건 포함 여부를 확인

### 병원
- [ ] 의료기관 개설허가·신고와 용도·소방 기준을 확인
- [ ] 의료장비·비품의 소유자 및 매각 포함 여부를 확인
- [ ] 입원환자·임차운영자 명도와 원상복구 비용을 확인

### 아파트상가
- [ ] 단지 내 상가의 구분등기·대지권·전용면적을 확인
- [ ] 단지 세대수, 동선, 독점업종 약정과 실제 상권을 확인
- [ ] 관리비 체납, 임차인 대항력과 권리금 위험을 확인

### 창고
- [ ] 건축물대장상 창고 용도와 실제 사용상태를 확인
- [ ] 냉동·냉장 등 특수설비와 보관물품의 소유자를 확인
- [ ] 대형차 진입로·하역공간·불법 가설창고 여부를 확인

### 목욕시설
- [ ] 목욕장업 신고와 위생·소방·보일러 검사 승계 여부를 확인
- [ ] 대형 설비·배관·수질시설의 노후도와 포함 여부를 확인
- [ ] 영업중단, 철거·원상복구 및 유치권 위험을 확인

### 콘도(호텔)
- [ ] 관광숙박업·휴양콘도미니엄 인허가와 운영계약을 확인
- [ ] 회원권·분양계약·위탁운영 및 관리비 부담을 확인
- [ ] 객실 비품과 공용시설 지분의 매각 포함 범위를 확인

### 운동시설
- [ ] 체육시설업 신고·등록과 건축물 용도 적합성을 확인
- [ ] 운동기구·락커·샤워시설 등 동산의 매각 포함 여부를 확인
- [ ] 회원권·선수금 반환채무와 원상복구 비용을 확인

### 휴게시설

- [ ] 주방·환기·소방시설과 영업시설의 소유관계를 확인
- [ ] 임차인 대항력, 권리금 및 원상복구 범위를 확인

### 노유자시설
- [ ] 노인·아동·복지시설 신고와 시설기준 충족 여부를 확인
- [ ] 입소자 보호 및 운영자 퇴거·시설 이전 계획을 확인
- [ ] 보조금·시설장비·소방설비의 소유와 승계 여부를 확인

### 자동차관련시설
- [ ] 정비공장·주차장·세차장 등 세부 용도와 인허가를 확인
- [ ] 리프트·세차기·검사장비 및 폐유시설의 포함 여부를 확인
- [ ] 토양오염·폐기물·원상복구 비용을 확인

- [ ] 데크·카라반·수영장·정화조 등 제시외 시설을 확인
- [ ] 진입로·주차·상하수도와 불법 증축 위험을 확인

### 교육시설

- [ ] 교구·시설물의 소유 및 매각 포함 여부를 확인
- [ ] 수강료 선수금, 임차인 대항력과 원상복구 범위를 확인

- [ ] 냉장·화장·봉안 설비의 소유와 검사상태를 확인
- [ ] 운영자 점유, 계약관계와 원상복구 비용을 확인
