# 물건별 초보 체크리스트

> 물건종류를 처음 검토하는 담당자가 바로 따라갈 수 있도록, 큰 묶음보다 실제 선택 가능한 물건종류 중심으로 나눈 체크리스트입니다.
> 사용법: 먼저 **공통 기본점검**을 확인한 뒤, 해당 물건종류 체크리스트를 추가로 확인합니다. 기준일 2026-06-30.

---

## 공통 기본점검

- [ ] 사건번호, 법원, 물건번호, 주소가 입력 자료와 일치하는지 확인
- [ ] 등기부, 매각물건명세서, 현황조사서, 감정평가서가 모두 확보되었는지 확인
- [ ] 말소기준권리가 무엇인지 확인
- [ ] 말소되지 않고 인수될 수 있는 권리가 있는지 확인
- [ ] 임차인 전입일, 확정일자, 배당요구일을 서로 대조
- [ ] 선순위 임차인 보증금 인수 가능성 확인
- [ ] 소액임차인 최우선변제, 당해세, 임금채권 등 배당에 먼저 빠질 금액 확인
- [ ] 감정가, 최저가, 예상 배당, 선순위 채권, 집행비용을 대조
- [ ] 취하 가능성, 무잉여 가능성, 입찰 전 변경 가능성 확인
- [ ] 점유자, 명도 난이도, 예상 명도비용 확인
- [ ] 관리비, 공과금, 체납금, 미납 부담이 있는지 확인
- [ ] 사진, 지도, 현장 상태가 감정평가서 내용과 맞는지 확인

## 아파트

### 일반 아파트
- [ ] 동, 호수, 면적, 대지권 비율이 등기부와 감정평가서에서 일치하는지 확인
- [ ] 선순위 임차인 또는 전세권이 있는지 확인
- [ ] 관리비 체납 여부와 공용부분 관리비 인수 가능성 확인
- [ ] 재건축, 리모델링, 안전진단, 조합 설립 여부 확인
- [ ] 대지권 미등기 또는 토지 별도등기 여부 확인
- [ ] 동일 단지 최근 실거래가와 최저가 비교
- [ ] 전입세대 열람 내용과 현황조사서 점유자가 일치하는지 확인

### 재건축·재개발 아파트
- [ ] 정비구역 지정 여부 확인
- [ ] 조합원 지위 승계 가능 여부 확인
- [ ] 추가 분담금, 이주비, 청산 가능성 확인
- [ ] 권리가액, 종전자산평가액, 분양신청 여부 확인
- [ ] 매각 후 조합 승계 절차와 제한사항 확인

## 빌라·다세대·연립

### 다세대주택
- [ ] 호수와 전유부분 면적이 등기부, 건축물대장, 감정평가서에서 일치하는지 확인
- [ ] 선순위 임차인 보증금이 시세 대비 과도한지 확인
- [ ] 대지권 미등기, 대지권 비율 누락, 토지 별도등기 여부 확인
- [ ] 신축빌라인 경우 분양대금 미납, 임차권등기명령 여부 확인
- [ ] 불법 증축, 옥탑, 베란다 확장 등 위반건축물 여부 확인
- [ ] 관리비 또는 공동관리비 체납 여부 확인

### 연립주택
- [ ] 세대별 구분등기 여부와 실제 점유 호수가 일치하는지 확인
- [ ] 공동주택 관리비와 장기 체납 여부 확인
- [ ] 대지권 또는 토지 지분이 정상적으로 포함되었는지 확인
- [ ] 오래된 연립의 경우 누수, 구조, 주차, 진입로 문제 확인
- [ ] 선순위 임차인과 소액임차인 최우선변제 여부 확인

### 신축빌라
- [ ] 감정가와 실제 시세 차이가 큰지 확인
- [ ] 보증금이 매매가에 근접한 깡통전세 위험 확인
- [ ] 임차권등기, 전세사기 관련 기록, 다수 경매 여부 확인
- [ ] 대지권 정리 여부와 건축물대장 사용승인 여부 확인
- [ ] 분양대행, 시행사, 시공사 관련 미정산 위험 확인

### 다가구주택

- [ ] 전입일, 확정일자, 보증금, 배당요구일 기준으로 배당순위표 작성
- [ ] 인수되는 보증금이 있는지 세대별로 확인
- [ ] 소액임차인이 여러 명일 때 최우선변제 총액 확인

### 다중주택·고시원형 주택

- [ ] 입주자 수와 보증금 합계 확인
- [ ] 영업 인허가나 용도 위반 가능성 확인
- [ ] 명도 대상자가 다수인지 확인
