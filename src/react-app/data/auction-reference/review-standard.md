## A. 공통 권리분석 체크리스트

- [ ] 등기부 인수권리(선순위 가처분·가등기·전세권·지상권·예고등기) 확인

- [ ] 현황조사서·감정평가서·집행관 조사로 점유·제시외·현황 확인
- [ ] 체납 관리비·당해세·조세채권(배당 순위) 확인

## B. 특수상황별 법률 검토 기준

### B-1. 선순위임차인/대항력  `747건`

**① 법률자문의견**

> 말소기준권리보다 전입(또는 전세권 설정)이 앞서 대항력 있는 임차인이 있거나, 배당요구를 하지 않아 보증금이 낙찰자에게 인수될 가능성이 있는 물건. 말소기준권리일과 전입일·확정일자·배당요구 종기·배당요구 유무를 대조해 인수 여부를 결론.

**② 보충 법률의견 (현행 법령·판례)**

> 주택임대차보호법 3조 — 인도+전입신고 다음 날 0시 대항력. 말소기준보다 앞서면 매수인이 임대인 지위 승계(보증금 인수), 확정일자(3조의2)로 우선변제. 배당요구한 대항력 임차인은 미배당 차액만 인수. 상가는 상가임대차보호법 별도. 채무자·가족이면 대항력 부정 여지.

**③ 대처**

- 전입일자 ≥ 말소기준권리일 → 대항력 인정, 미배당 보증금은 낙찰자 인수
- 대항력 임차인이 배당요구·전액배당이면 인수 0, 일부배당이면 차액 인수
- 배당요구 종기 내 배당요구·확정일자 순위 확인
- 보증금 인수 가능액을 입찰가에서 차감

**④ 조사 체크리스트**

- [ ] 전입세대열람·주민등록 전입일 vs 말소기준권리 설정일 비교
- [ ] 확정일자·배당요구 종기·배당요구서 제출 여부 확인
- [ ] 임차보증금 vs 예상배당액 → 인수 차액 계산
- [ ] 임차인이 채무자/소유자 가족(무상거주)인지 확인
- [ ] 상가는 환산보증금·사업자등록일 별도 확인

### B-2. 위반건축물  `860건`

**① 법률자문의견**

> 건축물대장에 위반건축물 등재 또는 무허가·무단증축·용도변경이 있는 물건. 위반 형태와 양성화 가능성을 검토하고 시정명령·이행강제금·고발 위험을 안내.

**② 보충 법률의견 (현행 법령·판례)**

> 건축법 79·80조. 2019.4.23 개정(법률 16380호)으로 이행강제금 총 5회 제한 폐지 → 시정 시까지 연 2회 이내 반복 부과(상한 없음). 위반 등재 시 영업허가 등 인허가·대출 제약. 이행강제금·철거비는 낙찰자 부담.

**③ 대처**

- 건축법 79·80·108·113조 — 시정 전까지 이행강제금 반복 부과
- 양성화 가능 여부를 관할 건축과에 확인, 불가 시 철거·원상복구 비용
- 불법 용도변경은 별도 과태료
- 이행강제금·철거비를 입찰가에 반영

**④ 조사 체크리스트**

- [ ] 건축물대장 위반 등재·내용·면적 확인
- [ ] 현황(증축·옥탑·근생→주거) vs 대장 대조
- [ ] 관할 건축과 이행강제금 이력·금액·양성화 가능성 문의
- [ ] 제시외 건물의 위반 여부 확인
- [ ] 멸실 건물 멸실신고 여부 확인
- [ ] 이행강제금 시정 시까지 연 2회 반복 부과(2019 횟수제한 폐지) — 누적 부담 추정  *(법령·판례 보강)*
- [ ] 위반 등재로 영업허가 등 인허가·대출 제한 여부 확인  *(법령·판례 보강)*

### B-3. 제시외건물  `275건`

**① 법률자문의견**

> 등기·대장에 없으나 현장에 존재하는 건물. 부합물·종물인지 독립 타인소유 건물인지 구분, 매각 포함/제외 및 법정지상권 연계 검토.

**② 보충 법률의견 (현행 법령·판례)**

> 부합물·종물이면 경매목적물 포함 취득(민법 100·358조). 독립 별개 건물(타인소유)이면 매각제외·법정지상권/철거 분쟁. 감정평가서 포함/제외 표기와 매각물건명세서 확인. 미등기·무허가는 위반건축물 리스크 병행.

**③ 대처**

- 부합물·종물이면 일괄 매각·평가 → 낙찰자 취득
- 독립 건물(타인 소유) 매각제외면 법정지상권·철거 이슈
- 감정평가서 제시외 평가 포함 여부·금액 확인
- 제시외가 위법건물이면 위반건축물 동시 검토

**④ 조사 체크리스트**

- [ ] 감정평가서 제시외 표시(포함/제외)·사진·면적 확인
- [ ] 제시외 소유관계(채무자 vs 제3자) 추정
- [ ] 매각물건명세서 제시외 포함/제외 문구 확인
- [ ] 제외 시 법정지상권 성립·철거 가능성 검토

### B-4. 법정지상권  `202건`

**① 법률자문의견**

> 매각 제외된 지상물(견사·컨테이너·가설건축물)이 토지 위에 있어 토지만 낙찰 시 (관습법상) 법정지상권 성립 여지가 있는 물건.

**② 보충 법률의견 (현행 법령·판례)**

> ① 저당물 경매로 토지·건물 소유자가 달라지면 건물소유자는 민법 366조 법정지상권 취득. ② 매매·증여·강제경매로 분리 시 관습법상 법정지상권 — 대법원 2022.7.21. 2017다236749 전합이 유효 재확인. ③ 처분 당시 동일인 소유·철거특약 부존재가 요건. 토지만 낙찰 시 지료청구, 2년분 연체 시 지상권소멸청구.

**③ 대처**

- 토지·건물 동일소유→경매분리 시 건물소유자 법정지상권 취득
- 토지만 낙찰 시 지료 청구, 2년분 연체 시 지상권소멸청구·철거 검토
- 무허가·가설건축물이면 양성화·철거 협의로 해소
- 지상물 제약을 입찰가에 보수적 반영

**④ 조사 체크리스트**

- [ ] 매각물건명세서·감정평가서 매각제외 지상물 목록·동수 확인
- [ ] 제외 건물 소유자(채무자 동일여부) 조회
- [ ] 건물 신축시점이 저당권 설정 이후인지 확인
- [ ] 지료 산정 기준(임료·공시지가) 파악
- [ ] 처분 당시 토지·건물 동일인 소유였는지(핵심요건) 등기 확인  *(법령·판례 보강)*
- [ ] 지료 미정 시 지료청구소송, 2년분 연체 시 지상권소멸청구 검토  *(법령·판례 보강)*

### B-5. 맹지  `171건`

**① 법률자문의견**

> 도로에 접하지 않은 토지. 건축·진입 제약이 커 가치가 낮음. 진입로 확보·주위토지통행권·입찰가 감액을 검토.

**② 보충 법률의견 (현행 법령·판례)**

> 건축법 44조 — 대지는 2m 이상 도로 접도 의무. 맹지는 원칙 건축 불가. 민법 219조 주위토지통행권으로 확보 가능하나 범위·지료 다툼. 토지사용승낙·구거/도로 점용허가로 진입로 확보해야 건축허가 가능.

**③ 대처**

- 진입로 확보: 인접토지 사용승낙·매입, 주위토지통행권 소송, 구거 점용허가
- 건축 시 도로 지정·접도요건 충족 확인
- 맹지는 인근시세 대폭 감액 또는 산정 제외
- 현황도로의 법정도로 여부 확인

**④ 조사 체크리스트**

- [ ] 지적도·도로현황 접도 여부, 현황 통행로·폭 확인
- [ ] 인접 필지·국유지·구거 소유관계·통행 가능성
- [ ] 건축 인허가 부서에 맹지 건축가능 문의
- [ ] 주위토지통행권 확보 비용·가능성 검토

### B-6. 농지취득자격증명  `167건`

**① 법률자문의견**

> 지목이 전·답·과수원 등 농지인 물건. 매각결정기일까지 농취증 제출 필요, 미제출 시 매각불허가·보증금 몰수 위험.

**② 보충 법률의견 (현행 법령·판례)**

> 농지는 매각기일+1주일인 매각결정기일까지 농취증 제출, 미제출 시 매각불허가+보증금 몰수(배당 산입) 가능. 2022.5.18 개정 농지법으로 발급심사 강화(농지위원회 심의). 불법전용은 원상복구계획서로 발급 가능.

**③ 대처**

- 매각결정기일까지 농취증 제출 — 미제출 시 보증금 몰수 가능
- 불법전용 농지는 원상복구계획서+낙찰영수증으로 발급
- 비거주자·일정면적 이상은 농지심의위 심의
- 관정 포함 시 소유권 이전 후 30일 내 승계신고

**④ 조사 체크리스트**

- [ ] 지목·현황 농지 여부, 농취증 발급·심의 대상 관할 확인
- [ ] 불법전용 부분 원상복구계획 수립 가능 여부
- [ ] 농지심의위 개최일과 매각결정기일 선후 확인
- [ ] 관정·지하수 승계신고 대상 여부
- [ ] 농지전용 가능성·전용부담금 파악
- [ ] 매각결정기일 내 발급 가능한지 — 미제출 시 보증금 몰수 위험 사전 확인  *(법령·판례 보강)*
- [ ] 2022 개정 농지법 농지위원회 심의 대상·소요기간 확인  *(법령·판례 보강)*

### B-7. 지분(공유)매각  `117건`

**① 법률자문의견**

> 공유 부동산 일부 지분만 매각. 공유자 우선매수권, 공유물분할이 핵심.

**② 보충 법률의견 (현행 법령·판례)**

> 민사집행법 140조 — 공유자는 매각기일까지 보증 제공·최고가 동일가격으로 우선매수 신고, 신고 시 그 공유자에게 매각허가. 남용은 매각불허가 사유, 실무상 1회 제한 다수. 낙찰 후 공유물분할청구(민법 268·269조).

**③ 대처**

- 공유자 우선매수권 — 낙찰받아도 공유자 우선매수 시 상실
- 낙찰 후 공유물분할: 협의→현물/대금분할 소송
- 지분만으로 사용·수익·담보 제약 → 보수적 입찰
- 집합건물 토지공유지분 별도등기 근저당은 매각 소멸

**④ 조사 체크리스트**

- [ ] 공유자 수·지분·관계·우선매수 가능성 확인
- [ ] 공유물 사용현황(점유·수익자) 확인
- [ ] 현물분할 가능 형상인지 검토
- [ ] 집합건물 대지권·토지별도등기 소멸 여부 확인
- [ ] 공유자 우선매수는 매각기일까지(실무상 1회) — 낙찰 무효화 가능성 감안  *(법령·판례 보강)*
- [ ] 낙찰 후 공유물분할 시나리오·기간 검토  *(법령·판례 보강)*

### B-8. 유치권  `108건`

**① 법률자문의견**

**② 보충 법률의견 (현행 법령·판례)**
