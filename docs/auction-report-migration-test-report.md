## 실데이터 테스트 대기

아래 항목은 마이옥션 실제 로그인과 사건 페이지 접속이 필요하므로 테스트 계정과 사건 URL 수령 후 진행한다.

### 브리핑자료
- 사건 URL 1개 입력
- 마이옥션 로그인 성공
- 사건 기본정보 파싱
- 물건개요 입력
- 담당자 종합의견 입력
- 문서 이미지 삽입
- 강제집행 예상비용표 삽입
- PPT 저장
- PDF 다운로드 가능 여부 확인
- 다운로드 이력 저장 확인

### 권리분석 보증서
- 사건 URL 1개 입력
- 임차인 없음 사건 테스트
- 임차인 있음 사건 테스트
- 예상배당표 기준 무잉여 판단 테스트
- 취하 가능성 판단 테스트
- 특이사항 반영 테스트
- PDF/PPTX 저장 확인
- 다건 URL 배치 테스트
- ZIP 다운로드 확인
- special 권한이 아닌 사용자의 접근 차단 확인: 통과

## 실테스트에 필요한 입력
- 마이옥션 테스트 아이디
- 마이옥션 테스트 비밀번호
- 브리핑자료용 사건 URL 1개
- 권리분석 보증서용 임차인 없음 사건 URL 1개
- 권리분석 보증서용 임차인 있음 사건 URL 1개
- 예상배당표 기준 무잉여 판단이 가능한 사건 URL 1개
- 취하 가능성 판단이 가능한 사건 URL 1개
- 특이사항 반영 확인이 가능한 사건 URL 1개
